**Database for Vessels that have committed IUU fishing**

Represents a user perspective of a database that contains reported Vessels who have committed IUU Fishing. The user can filter out vessels by different specifications like company, vessel types, violation counts etc. There are also options for sorting the data, adding data, and executing build in functions to perform quick calculations. Users can also save their query result and also export it to other collaborators for further analysis.
